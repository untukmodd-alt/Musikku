# 🎵 MUSIKKU PWA — Panduan Deploy

## Cara Deploy ke Netlify (GRATIS, 5 menit!)

### Langkah 1 — Daftar Netlify
1. Buka https://netlify.com
2. Klik "Sign up" → daftar pakai email (gratis)

### Langkah 2 — Upload folder
1. Setelah login, klik **"Add new site"**
2. Pilih **"Deploy manually"**
3. **Drag & drop seluruh folder `musikku-pwa`** ke halaman itu
4. Tunggu sebentar... ✅ SELESAI!

### Langkah 3 — Dapat link
- Netlify akan kasih link gratis seperti:
  `https://musikku-abc123.netlify.app`
- Bagikan link itu ke teman-temanmu!

## Cara Pakai sebagai Admin

1. Buka link di Chrome HP
2. Masuk ke tab **⚙️ Admin**
3. Masukkan PIN: **1234** (bisa diganti nanti)
4. Upload lagu MP3 dari HP kamu
5. Lagu langsung muncul untuk semua orang! 🎵

---

## Cara Install di HP (Seperti APK!)

### Android (Chrome):
1. Buka link di Chrome
2. Ketuk tombol **⋮** (tiga titik) di pojok kanan atas
3. Pilih **"Tambahkan ke Layar Utama"**
4. Ketuk **"Tambah"**
5. ✅ Icon Musikku muncul di homescreen seperti APK!

### iPhone (Safari):
1. Buka link di Safari
2. Ketuk ikon **Share** (kotak dengan panah ke atas)
3. Pilih **"Add to Home Screen"**
4. Ketuk **"Add"**

## Catatan Penting

⚠️ Karena ini versi gratis tanpa database cloud:
- Lagu yang diupload admin tersimpan **sementara** di browser
- Kalau browser ditutup lama, perlu upload ulang
- Untuk versi permanen, perlu upgrade ke database cloud (Firebase dll)
